Assay Protocol

1.
Use plate layout widget to determine the number of wells to be used. Return unused wells to foil pouch
2.
Pipette 50 µL of standards/samples into corresponding well in duplicate
3.
Pipette 75 µL of assay buffer into non-specific binding (NSB) well
Will be used to determine non-specific binding of corticosterone conjugate to IgG antibody
4.
Pipette 50 µL of assay buffer into buffer control well
Will serve as a measure of maximal binding without any displacement of conjugate by sample
5.
Add 25 µL of the DetectX Corticosterone Conjugate to each well using a repeat pipettor
  • Syringes are labeled for solution they should be used with
  • Add syringe to pipettor
  • Insert into solution. Press button on back to uptake solution
  • Change volume by pressing menu button then ↑ ↓ arrows
  • Eject first aliquot into solution container
  • Press button on back once to eject approriate aliquot into each well
  • Pay attention to when the syringe needs to be refilled if using a full plate
  • When finished, press down arrow. Screen should say E - pressing button will now eject entire contents
  • Eject remaining solution back into container
  • Rinse syringe with H2O. Return syringe to bag
6.
Add 25 µL of the DetectX Corticosterone Antibody to each well EXCEPT the NSB wells using repeat pipettor
7.
Gently tap the sides of the plate. Cover with plate sealer
8.
Shake for 1 hour at room temperature
Incubation complete at {Enter Start Time}
9.
Aspirate the plate and wash each well 4 times with 300 µL washBuffer
Tap the plate dry on clean absorbent towels
10.
Add 100 µL of the TMB substrate to each well using repeat pipettor
11.
Incubate the plate at room temperature for 30 minutes WITHOUT shaking
Incubation complete at {Enter Start Time}
12.
Add 50 µL of the stop solution to each well, using the repeat pipettor
13.
Read the optical density generated from each well in a plate reader capable of reading at 450 nm
