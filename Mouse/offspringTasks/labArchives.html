This widget can be used to keep track of birth dates and tasks for dams and offspring based on the offspring date of birth.

When editing the page, you can select from a list of options which tasks you intend to complete for each litter.

The widget will print out the accompanying task based on the current post natal day. You can select which day you want to view tasks for when editing the widget

Note that this widget does NOT attempt to track preputial separation, vaginal opening, etc where actions depend on the individual offspring and will vary by days. For this, use the appropriate individual widget.
